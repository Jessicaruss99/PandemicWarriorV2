Pandemic Warrior
By Lucas Dickson and Jessica Russ

Description
Pandemic Warrior is a single person action adventure platformer game, available on PC and intended for teens. 
Throughout this game, the player acts as the protagonist, an unnamed knight. 
This knight has to travel through multiple realms defeating deadly zombies, collecting health potions to stay alive, 
all to get to the final realm where the cure is hidden to save the day.

How to Play
Use left arrow key to run to the left
Use right arrow key to run to the right
Use the up arrow key to jump over obstacles such as lava
Press Z to pull out your sword
Collect potions to increase your score, speed, size and add an additional life,
Kill zombies and viruses and make your way to the final room to find the cure and save everyone

Secret Shortcuts
Press Esc to end the game
Press CTRL + T to replay the tutorial
Press CTRL + R to restart the current room
Press CTRL + G to restart the game
Press CTRL + L to add a life
Press CTRL + N to go to the next room
Press CTRL + F to go to the win screen
